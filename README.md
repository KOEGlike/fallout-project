# Beat a Soup
Beat A Soup is a handheld game console where you fight Soup himself! It has a load cell, two buttons and two leds, you have to press down on the loadcell with a given force to deal damage to Soup, and press the buttons when theirs corresponding LED lights up to protect yourself. 

<img width="4032" height="3024" alt="Adobe Express - file" src="https://github.com/user-attachments/assets/fee1ddd6-ae84-42ae-8da2-e27be604ca46" />


## Features:

- Xiao esp32c3
- soup art on the back
- 1.8" tft 
- Arcade buttons
- hx711 loadcell amp
- 80mm loadcell 
- rust firmware 
- soup key cap for the loadcell
